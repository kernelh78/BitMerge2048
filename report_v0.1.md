# BitMerge: 2048 Spark — 개발 리포트 v0.1

> 작성일: 2026-03-15
> 버전: 0.1.0+1
> 플랫폼: Android (primary), iOS/iPadOS (지원 예정)
> Bundle ID: `com.spark.bitmerge2048`

---

## 1. 프로젝트 개요

**BitMerge: 2048 Spark**는 클래식 2048 게임에 전기 스파크 / 회로기판 컨셉을 더한 모바일 게임이다.
Flutter로 개발하여 Android·iOS·iPadOS 단일 코드베이스에서 동작한다.

### 핵심 컨셉

| 항목 | 내용 |
|------|------|
| 디자인 | 다크 배경 + 네온 블루/퍼플/시안/골드 발광 효과 |
| 타일 | 회로기판 패턴 오버레이, 숫자 크기별 색상·글로우 자동 조절 |
| 승리 연출 | "⚡ Spark Ignited!" + 파티클 필드 + 아르페지오 효과음 |
| 게임오버 | "SYSTEM HALT" 오버레이 + 시스템 셧다운 효과음 |

## 2. 현재 구현 완료 항목

### 2.1 게임 로직

- [x] 4×4 그리드 기반 2048 게임 코어
- [x] 4방향 스와이프 이동 (위/아래/좌/우)
- [x] 타일 합치기 및 점수 계산
- [x] 새 타일 랜덤 생성 (2: 90%, 4: 10%)
- [x] 게임오버 판정 (빈 칸 없음 + 인접 합치기 불가)
- [x] 2048 달성 감지 → 승리 오버레이
- [x] 계속하기 (2048 이후 계속 플레이)
- [x] 현재 점수 / 최고 점수 추적

### 2.2 UI / UX

- [x] 다크 모드 기본 테마
- [x] 타일 값별 네온 색상 시스템 (2 ~ 2048+)
- [x] 회로기판 패턴 오버레이 (CustomPainter)
- [x] 타일 팝업 애니메이션 (새 타일: elastic scale)
- [x] 타일 합치기 펄스 애니메이션
- [x] 점수 증가 플로팅 텍스트 (+N 표시)
- [x] 헤더: 로고 + 타이틀 + 음소거 버튼 + 새 게임 버튼
- [x] 새 게임 버튼 누를 때 360도 회전 피드백
- [x] 전체 화면 스와이프 감지 (보드 외 영역 포함)

### 2.3 사운드 이펙트

외부 사운드 파일 없이 Dart로 직접 PCM 파형 합성 (22050Hz, 16-bit mono WAV):

| 이벤트 | 사운드 | 길이 |
|--------|--------|------|
| 타일 이동 | 전자 스윕 (주파수 하강) | 55ms |
| 타일 합치기 | 전기 스파크 자지직 (배음 포함) | 110ms |
| 새 타일 생성 | 디지털 틱 (고주파) | 25ms |
| 2048 달성 | 전력 급상승 아르페지오 (C-E-G-C) | 680ms |
| 게임오버 | 시스템 셧다운 (주파수 하강) | 350ms |

- [x] 음소거 토글 버튼 (헤더에 위치)
- [x] WAV 생성은 별도 Isolate에서 처리 — 메인 스레드 차단 없음

### 2.4 기술 스택

| 항목 | 내용 |
|------|------|
| 프레임워크 | Flutter 3.41.4 / Dart 3.11.1 |
| 렌더링 | Impeller (Vulkan) |
| 사운드 | audioplayers 6.6.0 |
| 최소 SDK | Android API minSdk (Flutter 기본값) |
| 타겟 SDK | Android API 36 (Android 16) |

### 2.5 프로젝트 구조

```
lib/
├── main.dart
├── game/
│   ├── game_state.dart      # 게임 로직 (이동, 합치기, 승패 판정)
│   └── audio_service.dart   # 사운드 합성 및 재생
├── theme/
│   └── spark_theme.dart     # 다크 테마 + 네온 색상 시스템
└── ui/
    ├── game_screen.dart      # 메인 화면 + 스와이프 감지
    ├── game_board.dart       # 4×4 보드 렌더링
    ├── spark_tile.dart       # 회로기판 타일 + 애니메이션
    ├── score_board.dart      # 점수판 + 플로팅 델타
    └── overlay_screen.dart   # Spark Ignited! / 게임오버 오버레이
```

## 5. 빌드 방법

```bash
# 디버그 실행 (연결된 기기)
flutter run -d <device_id>

# 릴리즈 APK 빌드
flutter build apk --release

# 릴리즈 AAB 빌드 (플레이스토어용)
flutter build appbundle --release

# iOS 빌드
flutter build ios --release
```

APK 출력 위치: `build/app/outputs/flutter-apk/app-release.apk`
