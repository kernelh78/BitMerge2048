> 버전: 0.1.0+1
> 플랫폼: Android (primary), iOS/iPadOS (지원 예정)

---

## 1. 프로젝트 개요

**BitMerge: 2048 Spark**는 클래식 2048 게임에 전기 스파크 / 회로기판 컨셉을 더한 모바일 게임이다.
Flutter로 개발하여 Android·iOS·iPadOS 단일 코드베이스에서 동작한다.

### 핵심 컨셉

| 항목 | 내용 |
|------|------|
| 디자인 | 다크 배경 + 네온 블루/퍼플/시안/골드 발광 효과 |
| 타일 | 회로기판 패턴 오버레이, 숫자 크기별 색상·글로우 자동 조절 |
| 승리 연출 | "⚡ Spark Ignited!" + 파티클 필드 + 아르페지오 효과음 |
| 게임오버 | "SYSTEM HALT" 오버레이 + 시스템 셧다운 효과음 |

## 2. 현재 구현 완료 항목

### 2.1 게임 로직

- [x] 4×4 그리드 기반 2048 게임 코어

- [x] 타일 합치기 및 점수 계산
- [x] 새 타일 랜덤 생성 (2: 90%, 4: 10%)

- [x] 2048 달성 감지 → 승리 오버레이
- [x] 계속하기 (2048 이후 계속 플레이)
- [x] 현재 점수 / 최고 점수 추적

### 2.2 UI / UX

- [x] 다크 모드 기본 테마
- [x] 타일 값별 네온 색상 시스템 (2 ~ 2048+)

- [x] 타일 팝업 애니메이션 (새 타일: elastic scale, 220ms)
- [x] 타일 합치기 임팩트 애니메이션 (1.0→1.45→0.88→1.04→1.0 바운스, 300ms)
- [x] 타일 슬라이드 애니메이션 (ID 기반 위치 추적, easeOutCubic 130ms)
- [x] 점수 증가 플로팅 텍스트 (+N 표시)
- [x] 헤더: 로고 + 타이틀 + 음소거 버튼 + 새 게임 버튼

- [x] 전체 화면 스와이프 감지 (보드 외 영역 포함, 속도 기반 보완)

### 2.3 사운드 이펙트

외부 사운드 파일 없이 Dart로 직접 PCM 파형 합성 (22050Hz, 16-bit mono WAV):

| 이벤트 | 사운드 | 길이 |
|--------|--------|------|
| 타일 이동 | 틱-슉 클릭 + 주파수 스윕 (700→220Hz) | 75ms |
| 타일 합치기 | 노이즈 어택 + 4중 하모닉 전자 팡 | 220ms |
| 새 타일 생성 | 디지털 틱 (고주파) | 30ms |
| 2048 달성 | 전력 급상승 아르페지오 (C-E-G-C) | 680ms |
| 게임오버 | 시스템 셧다운 (주파수 하강) | 350ms |

- [x] 음소거 토글 버튼 (헤더에 위치)
- [x] WAV 생성은 별도 Isolate에서 처리 — 메인 스레드 차단 없음

## 5. 변경 이력

### 2026-03-15 — 애니메이션·사운드·입력 개선

#### 문제
1. 타일 이동이 끊겨 보임 (슬라이드 없이 순간이동)
2. 빠른 플릭 시 스와이프가 간헐적으로 미등록
3. 사운드가 임팩트 부족

#### 원인 및 수정

| 파일 | 원인 | 수정 |
|------|------|------|
| `game_state.dart` | `swipe()` 호출 시 모든 타일을 새 객체로 재생성 → ID 소실 | `_mergeLine()`에 이동 경로 추적(`moves`) 추가, 슬라이드 타일은 기존 ID 보존 |
| `game_board.dart` | `ValueKey('tile_${r}_${c}_$value')` — 위치가 바뀌면 새 위젯 생성, `AnimatedPositioned` 미작동 | `ValueKey(tile.id)` 로 변경, `state.tiles` 기반으로 렌더링 |
| `game_board.dart` | 커브·속도 밋밋함 | `Curves.easeOutCubic` 130ms로 경쾌한 슬라이드 |
| `spark_tile.dart` | 합체 팝 스케일 1.25, 밋밋 | 1.0→1.45→0.88→1.04→1.0 스프링 바운스 300ms |
| `audio_service.dart` | 볼륨 0.7, 파형 단조로움 | 볼륨 1.0, 슬라이드·합체 파형 재설계 (노이즈 어택 + 다중 하모닉) |
| `game_screen.dart` | `GestureDetector`가 Stack 뒤에 위치 → 앞 위젯에 차단 | GestureDetector를 Stack 바깥으로 이동, 속도(velocity) 기반 스와이프 보완 |
